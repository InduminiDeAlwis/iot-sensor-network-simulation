# IoT Sensor Network Simulation

This repository contains a small IoT sensor network simulation project skeleton.

Structure created:

- src/main/java/client
- src/main/java/server
- src/main/java/utils
- src/main/resources
- src/test/{server,client}
- data
- dashboard
- build/{classes,logs}

Use this skeleton as a starting point for implementing the simulation logic.
